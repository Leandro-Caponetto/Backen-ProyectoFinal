# Backen Ecommers
## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/Leandro-Caponetto/Backen-ProyectoFinal.git)
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
