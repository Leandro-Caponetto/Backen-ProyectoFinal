# Backen Ecommers
## Run Locally

Clone the project

```bash
  git clone https://github.com/Leandro-Caponetto/Backen-ProyectoFinal.git
```

Go to the project directory

```bash
  cd Backen-ProyectoFinal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
